# Lineac Text Editor 
## Description

This Qt C++ project provides a simple and robust text editor enabling multi tabs

## Features
	File -> New
	File -> Open
	File -> Save
	File -> Save As
	File -> Quit
	Cursor position
	Multi Tabs management
	Edit -> Find
	Edit -> Find and Replace
	File -> Recent Files
